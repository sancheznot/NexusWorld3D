# Character models / Modelos de personaje

These files are **included in the template repository** so a fresh clone can open the character creator without 404 errors.

| File | URL | Purpose |
|------|-----|---------|
| `men/men-all.glb` | `/models/characters/men/men-all.glb` | Male player + animation clips |
| `women/women-all.glb` | `/models/characters/women/women-all.glb` | Female player + animation clips |
| `men/men_shape.png` | `/models/characters/men/men_shape.png` | Inventory UI silhouette (optional) |
| `women/women-shape.png` | `/models/characters/women/women-shape.png` | Inventory UI silhouette (optional) |

## Other character GLBs

NPC variants (`men_01.glb`, `women_01.glb`, etc.) and split animation files are **not** shipped by default. They stay under `public/models/` (gitignored). See `docs/REQUIRED_MODELS.md`.
